# Windows Server Active Directory Lab

Este projeto documenta a instalação e configuração completa de um ambiente
Windows Server para estudos de infraestrutura e gestao de identidades e acessos.

## Tecnologias utilizadas
- Windows Server 2019
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy (GPO)
- Microsoft Entra ID

## Objetivo
Criar um ambiente corporativo simulado com:
- Domínio Active Directory funcional
- Controle de usuários e grupos
- Políticas de segurança via GPO
- Integração com Entra ID (ambiente híbrido)
- Base para testes de MFA, Acesso Condicional, GPOs.
